# TODOLIST

***

## Планировщик задач


### Стек:
+ Python 3.11
+ Django 4.1.7
+ Postgres
+ Docker


### Установка
1. Клонируем репозиторий к себе на хост `https://github.com/DenisAhmedov/todolist.git`.
2. Устанавливаем зависимости командой `pip install -r requirements.txt`.
3. Накатываем миграции `manager.py migrate`.
